Client:
Parinishtaa Media Pvt. Ltd. (Easy Cabs, Sky Cabs)

Technologies Used:
Android, HTML5
Cloud Platform
Java, jQuery
The Problem:

The client wanted to build an In-cab entertainment system which could deliver rich content to In-cab displays with dynamic over the air updates delivered from the Cloud. The content needed to be transmitted and stored securely.


We were asked to build a Cloud based CMS allowing management of media & advertisements in cabs operating in all major cities across India.

Our Solution:

Byteridge architected and built an android based cloud solution to ensure high scalability. The Android based display application was configured with authentication for ensuring content security.


The fully automated system automatically delivers content and advertisements to the in-system storage via cloud whenever 3G connectivity is available and reports back tracking data on consumer impressions

IMPACT
Solution is currently installed in over 1000+ cab entertainment displays operating across major cities in India and interacting with the server in real-time. The system auto updates once the device is installed and can easily be managed without requiring a dedicated full time resource.

Client:
Swym Corporation

Technologies Used:
Google Android, Google Cloud
Messaging, Google Save-To-Wallet
The Problem:

Swym Corporation is a retail startup which required an Android application to complement their iOS and web apps. They needed a native Android application developed which worked similar to their existing applications on iOS and web with the same UX.


Byteridge was asked to create a native Android application which would integrate with their existing web-services.

Our Solution:

Byteridge developed a native Android application designed to be responsive to any screen size running the Android OS.


The application integrated well with their web-services and implemented Google Cloud Messaging, Google Save-To-Wallet APIs.

IMPACT
The responsive design of the native Android application allowed Swym Corporation to deploy to a majority of their target customers who use an Android smartphone or tablet.

Client:
HealthHihway, An Apollo Hospitals Company

Technologies Used:
Android, J2ME, LWUIT
Web Services
The Problem:

The client operates a health information network that enables health organizations to connect with each other to share data and conduct transactions between providers, employers, payers, TPA's, practitioners and patients.


Their patient population had limited PC+ Internet penetration and the Client wanted to use mobility to extend the reach of their health information network. We were asked to develop an application to enable mobile access for Patient health records and lab reports.

Our Solution:

The mobile app built by Byteridge enabled efficient and seamless interaction between the Hospital and Patients. Once registered, patients could use the app to access their health records and lab reports.


Any communications or notifications were automatically displayed on the app without the need for calling or visiting the Hospital. The app also enables sharing of the reports via email.

IMPACT
The mobile application has now been provided to patients and has successfully been adopted by many patients receive notifications on lab report status and to view their lab reports over mobile devices
